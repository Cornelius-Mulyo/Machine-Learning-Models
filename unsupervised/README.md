# ResNet50 CIFAR-10 Project

This project demonstrates transfer learning using ResNet50 on the CIFAR-10 dataset.

## Features
- Uses pretrained ResNet50
- Applies transfer learning
- Trains on CIFAR-10 dataset
- Evaluates model performance

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. View training results and accuracy

## Note
ResNet50 is designed for 224x224 images, so performance may be limited on CIFAR-10 (32x32).

## Future Improvements
- Resize images to 224x224
- Add data augmentation
- Fine-tune deeper layers
